# Amazon Sales Analysis

## Project Overview
This project focuses on analyzing Amazon sales data to understand sales performance, revenue trends, customer purchasing patterns, and business insights using Python-based data analysis techniques.

## Objective
- Analyze sales data to identify important trends and patterns.
- Understand revenue generation over different time periods.
- Perform exploratory data analysis (EDA) to extract meaningful insights.
- Visualize sales performance using graphs and charts.

## Tools & Technologies Used
- Python
- Jupyter Notebook
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Microsoft Excel

## Project Workflow

### 1. Data Cleaning
- Handled missing values.
- Removed duplicate records.
- Converted data types for analysis.

### 2. Exploratory Data Analysis (EDA)
Performed analysis on:
- Total Revenue
- Monthly Sales Trends
- Product Category Analysis
- Sales Distribution
- Correlation Analysis

### 3. Data Visualization
Created visualizations using:
- Bar Charts
- Line Charts
- Heatmap
- Other statistical plots

## Key Insights
- Identified sales trends across different months.
- Analyzed revenue contribution from various categories.
- Observed relationships between different sales-related factors.
- Extracted business insights from customer purchasing data.
